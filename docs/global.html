floodFill(board, start) → {number}

Counts how many board squares are reachable from a starting position using a breadth-first flood fill. Squares occupied by any snake body are treated as walls, as are positions outside the board boundaries. Used by the move logic to prefer directions that lead into larger open areas, helping the snake avoid trapping itself in a dead end.
Parameters:
Name Type Description
board Object The game board with dimensions and all snakes.
start Object The square to begin the flood fill from.
